Basic math in JS: numbers and operator

依旧和其他编程语言一样，js也有着内置的各种运算符和math等等。虽然并不是完全一样的，但是逻辑基本相同。

Number types

首先来讲一下数字类型，编程语言的数字系统有很多种，数字类型也有很多种。总的来说分为：

  • integers， 整数，可负可正。比如1，23，0，-6
  • Floating point numbers，小数，也就是含有decimal的数字，比如1.35
  • Doubles， 是一种特殊的floating point number，含有更多的小数位，更加精确

而数字类型就很多了，有正常的10进制，有二进制binary，八进制octal和十六进制Hexadecimal。但是好消息是，JS不在乎这些，上面的所有数字类型在JS里面等于一种：Number，并且使用方式基本相同

一个验证方法是使用console，定义两个变量，分别是小数和整数，使用typeof关键词来查询，将会发现是输出的都是number

Number methods

Numebr object是一个代表了所有JS内使用的标准数字实例*instance。有很多methods来用于操作数字，比如取整，取小数位等等。比如toFixed(2)就是取2位小数，只用将这个method加在变量，常量后面即可。 

      const manyDecimal = 1.23432546547475;
      console.log(manyDecimal)

      const twoDecimal = manyDecimal.toFixed(2);
      console.log(twoDecimal)

有的时候可能会遇到比较奇怪的问题，比如一个数字被定义成了字符串类型，这个时候直接使用Number() construct来改数据类型即可。例子如下: 

      let myNumber = '74';
      myNumber = Number(myNumber);

Arithmetic Operator

Operator Name Purpose Example
+ Addition Add two numbers together 5 + 1
- Subtraction Subtract right number from left number 2 - 1
* Multiplication multiply both numbers 2*8
/ Division Divides left number by the right 10/2
% Remainder (or modulo) Returns the Remainder left over from the division 8 % 3 (returns 2)
** Exponent use the base number to the exponent power 2 ** 3
