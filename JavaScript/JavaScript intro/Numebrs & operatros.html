Basic math in JS: numbers and operator

依旧和其他编程语言一样，js也有着内置的各种运算符和math等等。虽然并不是完全一样的，但是逻辑基本相同。

Number types

首先来讲一下数字类型，编程语言的数字系统有很多种，数字类型也有很多种。总的来说分为：

  • integers， 整数，可负可正。比如1，23，0，-6
  • Floating point numbers，小数，也就是含有decimal的数字，比如1.35
  • Doubles， 是一种特殊的floating point number，含有更多的小数位，更加精确

而数字类型就很多了，有正常的10进制，有二进制binary，八进制octal和十六进制Hexadecimal。但是好消息是，JS不在乎这些，上面的所有数字类型在JS里面等于一种：Number，并且使用方式基本相同

一个验证方法是使用console，定义两个变量，分别是小数和整数，使用typeof关键词来查询，将会发现是输出的都是number

Number methods

Numebr object是一个代表了所有JS内使用的标准数字实例*instance。有很多methods来用于操作数字，比如取整，取小数位等等。比如toFixed(2)就是取2位小数，只用将这个method加在变量，常量后面即可。 

      const manyDecimal = 1.23432546547475;
      console.log(manyDecimal)

      const twoDecimal = manyDecimal.toFixed(2);
      console.log(twoDecimal)

有的时候可能会遇到比较奇怪的问题，比如一个数字被定义成了字符串类型，这个时候直接使用Number() construct来改数据类型即可。例子如下: 

      let myNumber = '74';
      myNumber = Number(myNumber);

Arithmetic Operator

Operator Name Purpose Example
+ Addition Add two numbers together 5 + 1
- Subtraction Subtract right number from left number 2 - 1
* Multiplication multiply both numbers 2*8
/ Division Divides left number by the right 10/2
% Remainder (or modulo) Returns the Remainder left over from the division 8 % 3 (returns 2)
** Exponent use the base number to the exponent power 2 ** 3

上面的就是基本运算符，很简单。需要注意的是operator precedence是严格按照标准运算顺序来的。比如10+2/2+1等于12，而不是4。需要按照严格的数学逻辑来写算式

Increment/decrement operator

有的时候如果我想给一个变量加1或减1，那么可以使用increment or decrement operator，分别对应的是++，--。需要注意的是只能对于变量使用，不能类似3++ 

      let num = 1;
      num++;

需要注意的是，上面的代码若放在console内，会output出1，而这并不奇怪。这里面的逻辑是浏览器会先return current value，然后再increment。当然也有解决方案：将increment operator放在前面： 

      let num2=1;
      ++num;

Assignment operator

assignment operator是用于给变量赋值的运算符，最基本的是=，用左边的变量储存右边的值。不过还有以下几个常用的assignment operator

Operator Name Purpose Example Shortcut for
+= Addition assignment add the value on the right and return the new value x += 5 x = x + 5
-= subtraction assignment subtract value on the right and return the new value x -= 3 x = x - 3
*= Multiplication assignment multiples the right value and return the new value x *= 3 x = x * 3
/= Division assignment divided by the number on the right and return the new value x /= 5 x = x /5

基本就是以上内容了，很直观。

Comparison operators

当我们需要使用运行布尔测试的时候，我们就用到comparsion operator了。这种operator将会基于两个数字的条件来返回true/false

Operator Name Purpose Example
!== Strict equality Test if values are identical 6 == 3 + 3
!== strict non equality test if values are NOT identical 9 !== 5 + 4
< Less than test if left is greater than right 10 < 3
> Greater than test if left is greater than right 19 > 20
<= less than or equal to test if left is smaller than equal to right 3 <= 2

布尔值运算在和if，for循环判断结合后非常好用，可以用来判断各种情景
