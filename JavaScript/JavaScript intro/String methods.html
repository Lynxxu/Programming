Useful string methods

之前讲过了基本的Strings，现在将会深入讲一些JS的内置字符串操作。比如探测string长度，链接，断开，替换字符等等。

Strings as object

绝大多数的JS内容都是objects。当我们创建一个string的时候，比如const string = 'some stings'。这个时候变量就变成了一个string object instance。意思就是说我们可以使用各种properties和methods。这个时候可以参考mdn的string object page，然后就可以看见一堆可用的methods了。这种比较具体的信息可以当作词典使用，但是并不是每一种都百分百用得到，在这里就讲一下比较常用的

Length

使用length属性来查询一个字符串的长度。这个属性很好用，在很多时候可以用来根据长度排序，检测用户输入长度是否太长等等。比如下面的代码就会return 6 

      const browserType = "chrome";
      console.log(browserType.length);

Retrieving a specific character

另一个常用的东西是提取一个字符串内的指定character。可以使用square bracket notation，也就是[]和python差不多。从0开始数，下面的代码就会返回c 

      browserType[0];

假设我想要获得变量的最后一个字符，可以使用varName[varName.length-1]来提取，因为使用了长度减1，也就是最后一个字符的index

Test substring

有的时候如果想要探测一个字符串内是否包含小字符串（substring），可以使用include()method来返回true/false。 

      const names = 'ChalinJimOscar';

      if (names.include(Jim)) {
        console.log('found JIM');
      } else{
        console.long('No jim here');
      }

上面的逻辑就是一个简单的if判断变量names内是否包含jim，然后根据情况来返回回复。和这个类似的属性还可以探测是不是由某些substring开始或结束的，分别对应startsWith()和endWith()下面两个分别对应探测jim是否是开头或结尾， 

      const names = 'ChalinJimOscar';

      if (names.startsWith(Jim)) {
        console.log('Jim is no.1');
      } else{
        console.long('Jim is not No.1');
      }
    
      const names = 'ChalinJimOscar';

      if (names.endsWith(Jim)) {
        console.log('Jim is the last');
      } else{
        console.long('Jim is not last');
      }

Find position of substring in string

可以使用indexOf() method来查询一个substring在一个更大string的位置。这个method需要两个参数，一个是需要查询位置的substring在括号内，以及大string在前面 

      const bigString = 'chalin jim and oscar are groups of idiots';
      console.log(bigString.indexOf(idiots));

按照上面的代码，从0开始往后数到指定字符的位置，得到的返回就是35。需要注意的是，如果指定的substring不在范围内，那么将会返回-1

indexOf() method还可以多接受一个数字值。若不加上它，那么将会直接从0开始算到第一次出现。而如果加上一个数字值，则将会从数字指定的位置开始搜索。 

      const bigString = "chalin jim and oscar are groups of idiots idiots";
      const firstOccurrence = bigString.indexOf("idiots");
      cosnt secondOccurrence = bigString.indexOf('idiots', firstOccurrence + 1);

      console.log(firstOccurrence);
      console.log(secondOccurrence)

上面的代码块就会返回两个idiots分别出现的位置，后面可以用相同的逻辑继续写下去

Extracting substring from string

另外一个比较常用的method是slice()，依旧和python比较类似，可以接受两个值：第一个值指定开始切片的位置，第二个值指定停下的位置（不包含该位置的值）。若直接忽略第二个值，则会从第一个值开始截到最后一个字 

      const browserType = "chrome";
      console.log(browserType.slice(1, 2));
      console.log(browserType.slice(1));

上面两个slice分别会返回h和hrome。注意第一个slice是接受1，2的，意思是从index = 1处开始，到index=2处结束且不包含index =2，所以我们只获得了一个字符

Changing cases

在很多时候可以使用toLowerCase()和toUpperCase()调整字符串的大小写。这个用法在用户输入一些大小写不敏感的内容时很重要。例子如下： 

      const messedCap = 'SOme RandOM caPtaliZed WorDs'
      console.log(messedCap.toLowerCase());
      console.log(messedCap.toUpperCase())

Replacing parts of string

使用replace()和replaceAll()来将字符串内的某些部分指定换成其他内容，前者只会更换第一次出现的而后者会将全部符合的都换掉。 

      let sample = 'chalin is messing up with jim's network with oscar';
      const updatedSample = sample.replace('jim', 'william');

      console.log(sample);
      console.log(updatedSample);

就像上面的例子，replace和replaceAll接受两个值，一个是需要替换的值，第二个是替换成的值。需要注意的是使用这个method将不会对string造成影响，而是直接返回新的string，所以想要储存变化，需要重新赋值
