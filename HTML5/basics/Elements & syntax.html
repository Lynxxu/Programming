This is a paragrah 这句话是用<p class="foo">This is a paragraph</p&g>写出来的

第一个代表start tag,中间的class = "foo"为attribute和value,中间的是content，最后<p/p>为end tag，标志结束段落

在HTML内，tag为由尖括号括住的东西，标志着element的开始，而element就类似是语句了，是collection of tag, attribute and everything in between，不过这并不严禁，tag和element是可以互相交换interchangable使用的

HTML element types:
HTML的元素分为block level和 inline level, block level代表类似标题<h1 - h6>，段落<p>等需要整数位置的block，而inline element将会在行内结算，比如bold,emphasize与highlight
