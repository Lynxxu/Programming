Attribute

Definition:

Attribute为element的additional properties.比如<img src ="路径">内的src="路径"就是attribute name pair。下面我将展示一张阿符睡觉的图片并且设为的1920 x 960的长宽比,并且在无法展示时展示“阿符睡觉”的字样

代码为：<img src="D:\Programming\Imgs\阿符睡觉.jpg" width="960" height="1920" alt="阿符睡觉">，这里就有4个attribute分别代表图源，宽长和alternative display（即图片消失时展示的）

也有一些只有name的attribute，一般称为Boolean attribute，比如input，可以作出各种各样的responsives, 比如设置checkbox默认勾选等等:
checkbox checked
input box required
disabled submit button
readonly text input

General purpose attribute/global attribute

也有部分attribute为全局的，也就是不论什么element都可以用，比如:

id attribute,类似给一个变量，全局有且只能有一个，代表了一个element的unique identifier

还有class，这个就和py等语言的class一样，可以套用在多个element内，注意，一般来说class和id都要存在header内

以及title,这里说的title是指element的title，可以将鼠标指针附在上面后浮出字样类似这样

Style 则是和css有关的，设置元素的风格，比如色彩，字体，border等等，比如这句话就使用了style
