Head and metadata

Head

Head is the content of %lthead%gt element, it is not displayed on the page. It contains the metadata of the document.


Metadata in head

Title: There are two titles in a HTML document. One stands for the title as a metadata(in the head), which shows the title as the name for the tab. Another title is the title attribute in element, it shows the content when cursor is on the item.

Meta element: The meta element offers us a way to add metadata into a document. For example the <meta charset="utf-8"> element specifies the character encoding of the document. 当我输入像中文等语言时，因为utf8内包含了他，我们才不会得到一堆乱码，日本語は同じです。


Some examples of meta data

许多meta元素包含了name和content的attribute.name代表的是metadata的种类，包含了什么信息。content则是actual metadata。
当我们写description时，description其实是包含在了搜索引擎界面的，具体见MDN website

Other metadata

一个site肯定还有其他的metadata。比如我分享某个页面到qq或者其他social media上面，那个页面肯定不只是一个网址，而是一个包含了网页，简介的精美链接。这便也是一种metadata。 facebook和twitter均有他们自己的proprietary metada protocol.

Adding icons to the site

当我们创建网页时，可以选择一个favicon(favorite icon)也就是收藏夹里面的图标。
加入favicon的方式是：将icon存在html的相同目录，使用以下代码加入html内:
<link rel="icon" href="favicon.ico(icon名）" type="image/x-icon">

Setting primary language for the page

可以给页面设置主要语言，设置主要语言可以使得搜索引擎更有效的找到并display你的页面，当然，也可以设置subsection语言，比如这个文档有中日英三语，可以设置两个subsection

主语言将在head里面使用<html lang="en-US">元素，而subsection则可以加入<span lang = "zh">这是中文</span>来穿插其他语言，这里语言代码使用的是ISO 639-1标准

CSS and Javascript in HTML

在HTML内，我们会使用CSS来装饰页面，用javascript带来交互元素。而这两个都将在head内加入。
CSS将使用<link>加入，类似<link rel="stylesheet" href="my-css-file.css">。link元素必须在head内，两个attribute各代表了link的类型（stylesheet）和其路径

Javascript将使用<script>加入，并且也在head内。而script有两个attribute，一个是src，代表JS的文件路径。另一个则是defer，Boolean attribute，他使浏览器延后解析JavaScript文件（等html全parsing完）以避免奇奇怪怪的error。代码为：<script scr="my-js-file.js" defer> </script>
