Global requirements

需要注意的几个全局变量需求是，首先需要有html 5的doctype和一个responsive 的meta tage，也就是VS code自带的一大段开头： 

    
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
</html>

然后，对于css的box-sizing来说，bootstrap需要使用box-sizing，而不是borderbox或者content-box。
