Describing the UI

React 为一个JS库，专门用于渲染UI。React可以将许多小的units，比如buttons，images，texts加在一起，变成reusable nestable components。我将在这个页面学习react component的创建，定义和显示等操作。

First component

React applications are built from small isoalted pieces of UI called components. A react component is a JS function which can be changed, it can be small as a buttonm and large as an webpage.

Using HTMl, markup structures like a ordered list can be created, combined with css and JS. React would combine all the css, js and HTML into custom components that are resulable UI elements. Some open source communitites already have thousnads of components to be used like MUI

Define a component

Traditionally, web devs would write html, markup and add interaction via JS. It is now cumbersome and difficult if it were to be repeated thousands of times. React makes them reusable, but at the end of day, a React component is a JS funciton that you can sprinkle wiht markup. A block of code like below would generate 

        export default function Profile() {
          return (
            <img
              src="My waifu"
              alt="Noelle"
            />
          )
        }

Step 1: Export the component

export defautl is a standard JS syntax that allows the funciton to be exported to other fiels to be used. See later sections

Step 2: define function

Using function Profile() {}, this defines a JS function with name Profile. Note that a React component must start with CAPITALIZED letter, unlike regular functions
