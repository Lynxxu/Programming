Background and borders

css可以给html上背景，也可以改变border的style。更具体一点还可以有特殊的自定义背景比如渐变色。

Styling backgrounds

CSS有一个property叫做background的property，类似border等等，这也是一个shorthand form。这里先看单独的properties

Background colors

background-color属性定义任意一个element的背景色，此属性可以接受任意一个有效的颜色，并且将覆盖box里面的content和padding。比如这个段落以及这个span便是设置了一个color的。注意可以使用rgba来生成特定的颜色以及透明度，比如我想要一个半透明的黑色，则使用如下：rgba(0,0,0,0.5)

Controlling background-repeat

使用background-repeat来控制背景repeat behavior。具体有以下的几个属性：

  • no-repeat 关闭所有重复
  • repeat-x 只重复x轴
  • repeat-y 只重复y轴
  • repeat 默认，开启全部重复

Positioning the background image

虽然上面讲过了如何sizing，但是还有positioning。使用background-position来更改背景图片在box内的位置。一般来说他会接受两个值，分别代表horizontal以及vertical value。可以接受的值如下：

  • 使用关键词top和right，center等等来更改位置
  • 也可以使用length，percentage的形式，比如20px，10%等等来代表box的位置，从默认的左上角来算。
  • background-position也可以接受4个值，相对应从某个border开始的距离，比如background-position: top 20px right 10px对应的便是从上往下20px，从右往左10px的位置

Multiple background images

当然，也可以在一个box内添加多个背景，具体方法就是在background-image属性内添加多个值，使用逗号隔开。如此便会生成layered背景图片，顺序与图片的顺序相同，最后的图片在layer的最下面。

值得一提的是，每一个背景图片的properties也可以单独设置，在对应的特性里面使用逗号隔开，比如三个图片，便可以使用backgroundj-repeat:no-repeat, repeat-x, no-repeat;来分别对应第一二三张图片。以下的图片便是使用了多个layer得出的，第一张原神图片便是在第一位，第二个折木就是在第二个位置的图片。

Background attachment

还有几个关于背景的特性，说准确一点应该是背景与鼠标滑动的互动。使用background-attachement特性，接受以下几个值：

  • scroll:使元素的背景随着页面的滚动而滚动，即使content动了图片也不会动。背景图片固定位置。
  • fixed:使背景图固定，即使页面滚动，背景仍会留在位置上。
  • local:新的css值，会使得该元素的背景只与元素的content相关，整个页面的滚动不会影响，但是元素内的内容滚动时会随着滚动。

Accessibility consideration with backgrounds

使用背景图片时，一定要注意图片和文字的对比度，使所有用户都看得清楚。注意对于screenreader用户来说，背景图片是无法被解读的，所以所有的重要内容必须要在html的content内。

Borders

在之前提到过的box model内也包含一个重要的要素：border。具体加border的方式在box model内讲过了，简单来说就是可以使用简单的border来控制全部四个边，也可以单独控制某个边。

Rounded corners

可以使用border-radius，以及有关的longhand来调整圆角。使用该属性的方法有两个，如下：

border-radius:10px;

以及:
border-top-left-radius:10px;
分别对应了一次性控制全部边以及单独控制某两条边的角，甚至还可以调整某个边界的比重，比如：
border-top-left-radius: 30px 10px;
