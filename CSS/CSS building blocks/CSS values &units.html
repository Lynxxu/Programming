CSS values and units

CSS内的每一个property均有一种value type（或data type），这种type定义了该property可选择的values，比如<color>便是代表了一切颜色，那么一个完整的定义颜色的property value pair便是color: <color>

Numbers, lengths, and percentages

Data Type Description
<integer> 任意whole number，比如1025，-12
<number> <number>代表了任意decimal number，可以是小数也可以不是，比如999，1.2，-1.1
<dimension> <dimension>是一个带单位<number>，比如45deg，5s，19em等等。<dimension>是一个大类，其下属的有<length>, <angle>, 以及<resolution>
<percentage> <percentage>代表了某个value的比例，比如50%。Percentage永远是相对与某个值的，比如一个元素的长度是其母元素的90%

Lengths

css很常见的数字类型numeric type是<length>，比如10px, 30em等等，但是除了这种以外，length还接受relative length

em与rem

em与rem为两种相对长度，一般来说最常见的就是这两个unit了。首先记得em为parent element的font size，而rem为root element的font size（在html内，一般为html），因为这种特性，em与rem在nesting时会有不同的特征。

假设我想写一个nested list，如果我们使用的是rem，则nested list的全部内容均将由root元素，一般为html的font size 决定，于是字体便不会由特别大的变化。但是当我使用em时，由于nested list会使用母元素的font size来决定大小，则nested em会一直增长，类似这样：

Percentages

在大多数情况下，percentage的使用方法和length一样，但是和em类似，percentgae永远是与某个元素相对的，比如我设置font-size为50%，则font size将会是该元素parent的50%

比较有意思的是，使用nesting元素时，假设使用的时percentage，那么结果其实会与使用em类似，因为都是根据parent element的某个特性来的。只不过em根据的是font size，而percentage会以对应的来调整。值得一提的是，并不是所有的属性都会支持percentage，有一些只会支持length属性。记得查一下allowed values是否含有percentge

Numbers

有部分属性只接受数字，即不带单位的数字， 一个例子为opacity，控制某个元素透明度的属性，该属性接受从0到1的数字，即从完全透明到完全不透明

Color

有很多办法可以定义某个想要的颜色，而这些办法有些是新的，有些是已经支持很久了的。但是一般来说，一种颜色是可以在css内广泛使用的。现代电脑支持256 x 256 x 256 = 16,777,216种颜色， 由rgb三种tunnel定义

Color keywords

<color> value type可以接受keywords，比如grey，greenyellow，blueviolet等等。

Hexadecimal RGB values

另外一种经常遇得到的color value type为hexadecimal codes，每个hex value由hash#开头，接着6个16进制的值，即从0到f。每两个值代表了一种颜色，因为16x16=256。则总共对应依旧对应的是256 x 256 x 256种颜色。类似例子有#1d3fd5, #c55da1

RGB ans RGBA

第三个加入color的方式为rgb和rgba，为一种定义了rgb的function。而rgba则是附带了opacity的定义。比如rgb(2, 150, 200)以及rgba(99, 100, 5, 0.3)以及没有设置透明度的rgb(199, 100, 15)

HSL and HSLA vlaues

除了前面几个方法之外，浏览器支持较少的便是HSL color model（比较老的IE一般不支持）。和rgb不同，hsl是使用色相，饱和度和明度来定义多种颜色的。具体来说为以下：

  • Hue:定义色相环上的某个颜色，可选值为*从0到360如下：
  • Saturation，即饱和度，色彩的鲜艳程度，0为透明，而100为满色
  • Lightness亮度，0为黑色而100为纯白
与rgba类似，HSLA也只是多了一个指定透明度的值。一般来说在一个project内使用哪个颜色选项都可以

Position

<position> value type代表了一系列2d坐标， 使用于放置item时的定位，比如background-position，可以接受keywords比如top, left, bottom,right, 以及center来对于box定位。

一般来说，一个position value是有两个值的，第一个代表horizontal坐标，另一个代表vertical坐标。若不填的话便默认center，比如以下的色相盘便是是d使用的position: left center来定义的位置，即水平方向是左边，vertical方向为中间

Strings and identifiers

以前学过了很多种关键词，比如top, black, center等等。这些关键词被称为identifiers而不是string。因为css认识这些词语，所以也不需要使用quotation将其框起来。

但是并不是所有时候都用的是identifiers。比如想要使用一段string来代表你的content时，可以使用quote来将其圈起来以代表string

Functions

最后一类value type为functions。function具体的定义就不在解释了，在css内也有functions，比如rgb() , url()等等。

类似传统编程语言的function有calc()，用来指定特殊的尺寸以及值，一般用于无法计算的情况。假设calc(20%+ 20px)便是parent element的20%的尺寸加上20px
