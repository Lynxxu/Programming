Overflowing content

一切内容在css内都是由box组成的，我们可以自定义box的大小尺寸以及特性，但是假设文字所站的位置比box要大，则便有了Overflow

看上面的box，有许多内容因为box太小而出现在了外面，但是由于css的特性， 这些在box外的内容会显示出来，而不是消失。 CSS会avoid data loss，因此当有内容超出了box时，css默认你知道你在干嘛。但话虽如此，还是有办法可以调整overflow的，即使用overflow

The overflow properties

overflow为CSS的特性，用于控制css的overflow，默认值为visible，即看得见overflow的内容。除了visible之外，还有其他的几个value可选：

  • hidden：即将整个overflow的东西隐藏起来，注意隐藏起来后就看不见了。
  • scroll：浏览器将会使用scroll bar来展示overflow的内容，可以使用x，y等来指代具体的scroll方向，比如overflow-y:scroll
  • auto:自动决定，如果内容太多需要overflow，则会选择overflow。

Block Formatting Context

当使用scroll或者auto时，box会被改成Block Formatting Context格式，此时box内的内容无法突出box外，而box外的内容也无法进入box，而一切内容均会由scroll的形式保持完整。
