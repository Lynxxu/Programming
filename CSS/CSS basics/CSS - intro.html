What is CSS

Cascading Style Sheet, known as css, 用于网页的装饰以及html网页的元素。相对应的，html用于定义网页基本结构以及骨架。但是一个没有css的html网页会非常无聊，类似这样。因为默认html的style，即default style，是由浏览器定义的一个css style sheet，而当网页没有定义css时，浏览器会选择默认的style。

浏览器也被称为user agent，而一个document（文档，网页，一般由html，svg或者xml构成）会在浏览器内展示出来，css则可以更改文档内的元素的size，color，生成layout，甚至是生成简单动画。

CSS Syntax

CSS是rule-based language，在css内，我们通过定义某个特定的html的元素（组）的style来进行styling。

假设我想把大标题改成红色，5em尺寸（1em=16px），则这个特殊的css rule为： 

                    h1{
                        color:red;
                        font-size:5em;
                    }

上面的css rule由一个selector开始，选择某个需要的html元素。在这个例子里面我们选择的是h1

然后，property-value pair，即想要更改的属性以及更改的内容。比如color:red;，都会在大括号内包括，而这就是全部的css语法。

CSS modules

css的语言本身是由很多模块组成的，比如定义背景和边界会有一个专属的模块CSS Backgrounds and Borders。因此当寻找某个specific的css properties时，可以通过这种方式找到其所在的模块

CSS specification

类似html和其他的语言，css也有specification，即官方文档。定义一切关于css的东西。css是由W3C开发的。但是注意官方文档会十分bulky且麻烦，一般来说并不是学习语言的好路程，但是可以当作dict使用，毕竟其包含了底层机制。
