How is css structured?

This article specifies the structure of CSS

Applying CSS to HTML: 3 methods

As mentioned in css basics, CSS可以有三种加入方式：

  1. External:在html header内使用link 进行外部引用，最佳方式，一个css style sheet可以应用至多个文档，保证了一致性。
  2. Internal: 在html的header内写入css，仅对该文档生效
  3. Inline：在单独的元素内使用，非常麻烦，仅在特殊情况使用

Selectors

selectors具体的definition已经在前面讲过了。但是如果我们遇到了以下的情况，css会怎么抉择呢？ 

                .OrangeParagraph{
                    color:orange;
                }
                p{
                    color:blue;
                }
在以上情况，css会使用.OrangeParagraph，即class的css，因为其更specific，范围更小，这称为rule of specificty 
                p{
                    color:red;
                }
                p{
                    color:blue;
                }
而在以上情况，两个相同的selectors，不同的property-value pair，后面的会生效，这被称为rule of cascade

Properties and values

除了header以外，css还需要property-value pair，定义selector的特殊styel，以及style的属性。当property value pair被组合到了一起，比如:background-color:yellow;则他们被称为CSS Declaration，而花括号内的则被称为CSS Declaration block。当加上了selector，则他们被称为css rulesets

Functions

一般来说，values都是简单的关键词或者数值。不过有的时候他也可以是function

calc() Function

calc() function需要一个expression当parameter。这个expression可以是各种operators的组合，比如100%-30px就是将box填满，然后减去30像素的大小

100% -30px

Transform functions

另一个例子是Transform function，可以接受许多value，例如rotate()

@rules

@rules provides instruction on how css should perform，有些很简单，比如@import便是import一个stylesheet进另一个stylesheet。 

@import 'stylesheet.css'

又或者，@rules的有用的例子为@media。这个rule也被称为media queries，一般用于针对设备来调整网页/app的特征，比如这个网页便设定了基础背景色为白色，但是当屏幕尺寸小于50em （即900px）时，将会转为粉色背景。同理可以更改字体，图标等等。

Shorthands

有许多css selectors比如font，padding，可以有简化过的propertiesshorthand properties，比如padding:10px 20px 15px 10px相对应 

                padding-top: 10px
                padding-right: 20px
                padding-bottom: 15px
                padding-left: 10px

注释

HTML的注释为<!-- -->，而css的注释由/*开始，由*/结束，类似/*This is CSS comments*/，注释可以用于暂时disable部分代码，加详解等等。
