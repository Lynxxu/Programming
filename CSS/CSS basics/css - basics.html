Ways of adding CSS to document

总的来说，有三个向html内加入css的方法：

  1. External：在header内使用<link> element来引用外部的独立css stylesheet
  2. Internal：在header内使用<style>元素，在<style>内输入css
  3. inline：在特定的html元素内使用style attribute，让其只对该元素生效

Styling HTML elements

类似在css intro内，可以使用css来style某个html文档的全部特定元素。比如我想将这个文档内的所有段落都变成绿色，则是使用p selector，color property和green value 如下: 

            p{
                color:green;
            }
也可以使用多个selector，比如p，li

Override default behavior of element

对于一个结构完整的html文档来说，浏览器会有内置的默认styling，比如标题会是黑体大写的，超链接会有颜色等等。但是有的时候我们可以更改，或者覆写浏览器的某些默认css rules。而办法则是使用特殊的property，如下： 

            li{
                list-style-type: none;
            }
以上css指定的是li selector，即list里面的每个term，而list-style-type则是其marker的style。none value则是不设置marker。效果如下：

Adding a class

以上的办法会使html文档内的某个特定element全部生效，但是一般来说我们会需要部分element有特殊的style。而使用class便可以满足这个需求

使用class attribute加入想要更改的element，同时在css 内指定该特殊class的 style，则可以更改被指定了class的元素，比如我想要更改下一个段落为橘色的，则我需要在段落内加上class = "OrangeParagraph"，然后在css内加入以下代码： 

            .OrangeParagraph{
                color:orange;
            }

Orange Paragraph

记得使用css的指定class时，需要在前面加入一个句号，full stop character 以指定是使用class。另外可以使用类似p.OrangeParagraph来指定这个class只能限定于p element内。

Styling based on state

某些元素会有不同的state，比如一个超链接可以变色（点进去前vs进去后），而css可以对不同状态进行不同的styling。

举<a>为例子，假设我想设置超链接为蓝色，点击后变为绿色，然后当鼠标浮在上面的时候去掉下划线，则使用以下的语法： 

            a:link{
                color:blue;
            }
            a:visited{
                color:green;
            }
            a:hover{
                text-decoration:none;
            }

Combining selectors

有的时候可以将多个selector以特定形式结合，限定具体情况的styling。比如article p span{...}就是选择article内的p内的span。而h1 + ul + p{...}则选择h1后面的ul后面的p进行styling

复杂一点的则是全部加起来，比如body h1 + p .OrangeParagraph{...}则是body内，h1后面的p内的OrangeParagraph class
